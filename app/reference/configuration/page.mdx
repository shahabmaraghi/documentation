# Configuration Reference

Fine-tune the documentation site by adjusting these key files.

## `next.config.ts`

- Nextra is enabled via `export default withNextra({})`.
- Add additional Next.js options inside the `withNextra({ ...nextConfig })` call when needed.

## Sidebar structure

Edit `_meta.json` files to control ordering and visibility. Common properties:

- `title`: Display label in the sidebar.
- `display: "hidden"`: Hide a page from navigation.
- `display: "children"`: Hide a section heading while keeping its children visible.
- `type: "menu"`: Create a custom menu with explicit items.

## Environment variables

Use `.env.local` for secrets needed in the docs (API keys, analytics tokens). Prefix public values with `NEXT_PUBLIC_` so they can be read in the browser.
